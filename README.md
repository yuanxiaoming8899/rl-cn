<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">机器人</font></font><a href="https://www.roboticslibrary.org/" rel="nofollow"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">库</font></font></a><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">(RL) 是一个独立的 C++ 库，用于刚体运动学和动力学、运动规划和控制。</font><font style="vertical-align: inherit;">它涵盖空间矢量代数、多体系统、硬件抽象、路径规划、碰撞检测和可视化。</font><font style="vertical-align: inherit;">它被用于研究项目和教育，可在 BSD 许可证下使用，并可免费用于商业应用程序。</font><font style="vertical-align: inherit;">强化学习可以在许多不同的系统上运行，包括 Linux、macOS 和 Windows。</font><font style="vertical-align: inherit;">它使用 CMake 作为构建系统，可以使用 Clang、GCC 和 Visual Studio 进行编译。</font></font></p>
